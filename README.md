# GitHub User Activity CLI / Web App

A simple full-stack application that fetches and displays recent GitHub activity of a user using the GitHub API.

This project is part of the roadmap.sh backend projects.

Project Idea:
https://roadmap.sh/projects/github-user-activity


## Features

- Fetch recent GitHub activity
- Displays push, star, fork and issue events
- React frontend interface
- Express backend API
- Uses GitHub REST API
- Simple full-stack architecture


## Tech Stack

Frontend
- React
- Vite
- Axios

Backend
- Node.js
- Express.js
- GitHub REST API
